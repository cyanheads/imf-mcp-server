# 0.1.2 — 2026-06-05

## Added

- **`source` field** in `imf_query_dataset` and `imf_get_database` output — both `structuredContent` and `format()` now carry `Source: International Monetary Fund, <dataflow name>, https://data.imf.org/` per [IMF Copyright and Terms of Use](https://www.imf.org/en/about/copyright-and-terms) ([#1](https://github.com/cyanheads/imf-mcp-server/issues/1))
- **Data source section** added to README documenting the attribution requirement and format
