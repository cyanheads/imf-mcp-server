# 0.1.4 — 2026-06-06

## Fixed

- **`imf_dataframe_query`**: SQL validation (`invalid_sql`) now fires before the canvas-existence check, so DML/DDL statements are rejected even when `CANVAS_PROVIDER_TYPE=duckdb` is not set (#2)
- **`imf_query_dataset`**: Scale `"0"` (the no-op sentinel the IMF SDMX API emits when scale is absent) is now suppressed from the formatted `Series:` line; real scale values and units are unaffected (#3)
- **`imf_list_databases`**: When a substring filter matches nothing, a `notice` field is now populated in `structuredContent` (and rendered as a blockquote in `content[]`) explaining the applied filter and suggesting alternatives — previously the empty result was silent (#4)
